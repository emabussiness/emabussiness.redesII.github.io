Ejercicios Guiados en Packet Tracer

Ejercicio 1: Red Básica con 2 PCs

Objetivo: Configurar IPs manualmente y probar conectividad

Arrastra 2 PCs y un Switch 2960 al área de trabajo
Conéctalos con cables azules (FastEthernet)
Haz clic en PC1 → Desktop → IP Configuration
Selecciona Static y completa:
  • IP: 192.168.1.10
  • Subnet Mask: 255.255.255.0
Repite en PC2 con IP 192.168.1.20
Abre Command Prompt en ambas y escribe:
ping 192.168.1.10 # Desde PC2
Verifica: Debes ver "Reply from..."

Ejercicio 2: Crear 2 VLANs

Objetivo: Separar dispositivos en redes lógicas

Arrastra 1 Switch y 3 PCs
Haz clic en el Switch → Config → VLAN Database
Crea:
  • VLAN 10 (Nombre: "Oficina")
  • VLAN 20 (Nombre: "Ventas")
Asigna puertos:
  • FastEthernet0/1 → VLAN 10
  • FastEthernet0/2 → VLAN 20
Configura IPs:
  • PC1 (VLAN 10): 192.168.10.10/24
  • PC2 (VLAN 20): 192.168.20.10/24
Prueba comunicación con ping entre PCs
Resultado: No deben comunicarse (son redes diferentes)

Ejercicio 3: Servidor DHCP Automático

Objetivo: Configurar asignación automática de IPs

Arrastra 1 Servidor, 1 Switch y 2 PCs
Haz clic en el Servidor → Services → DHCP
Configura:
  • Pool: 192.168.1.100 to 192.168.1.200
  • Default Gateway: 192.168.1.1
  • DNS Server: 8.8.8.8
En las PCs → Desktop → IP Configuration → Selecciona DHCP
Abre Command Prompt y escribe:
ipconfig
Verifica: Debe mostrar una IP del rango 192.168.1.100-200

Ejercicio 4: Conectar 2 Redes con un Router

Objetivo: Permitir comunicación entre VLANs

Crea la topología del Ejercicio 2 (2 VLANs)
Agrega un Router y conéctalo al Switch con cable verde (GigabitEthernet)
Configura las interfaces del Router:
  • Gig0/0.10: 192.168.10.1/24 (VLAN 10)
  • Gig0/0.20: 192.168.20.1/24 (VLAN 20)
En las PCs, establece como Gateway:
  • PC1: 192.168.10.1
  • PC2: 192.168.20.1
Prueba ping entre PCs
Resultado: Ahora deben comunicarse

Ejercicio 5: NAT para Acceso a Internet

Objetivo: Configurar salida a Internet

Usa la topología del Ejercicio 4
Agrega un dispositivo Internet Cloud y conéctalo al Router
Haz clic en el Router → Config → NAT
Selecciona Interface: Gig0/1 (hacia Internet) y marca Overload
En Inside, agrega las redes:
  • 192.168.10.0/24
  • 192.168.20.0/24
Desde cualquier PC, intenta hacer ping a 8.8.8.8
Verifica: Usa el modo Simulation para ver el tráfico NAT
