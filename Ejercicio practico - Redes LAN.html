Ejercicio Práctico: Conexión de 3 Redes LAN

📌 Objetivo

Conectar 3 redes LAN diferentes (Oficina, Ventas y Almacén) mediante un router, utilizando solo la interfaz gráfica de Packet Tracer.

Topología Final

[PC0: 192.168.10.10] ↔ [Switch0] ↔ [Router] ↔ [Switch1] ↔ [PC1: 192.168.20.10] ↳ [Switch2] ↔ [PC2: 192.168.30.10]

🛠 Materiales Necesarios

  • Dispositivos:
    • 1 Router (2911)
    • 3 Switches (2960)
    • 3 PCs (una para cada red)
    • Cables azules (FastEthernet)
  • Direccionamiento:
    • Red Oficina: 192.168.10.0/24
    • Red Ventas: 192.168.20.0/24
    • Red Almacén: 192.168.30.0/24

🔧 Configuración Paso a Paso (100% Gráfico)

Paso 1: Armar la Topología

1

Arrastra los dispositivos al área de trabajo:

  • 1 Router 2911 (central)
  • 3 Switches 2960 (uno para cada red)
  • 3 PCs (una conectada a cada switch)
2

Conecta los cables:

  • Usa cables azules (FastEthernet):
    • Router Gig0/0 → Switch0 (Oficina)
    • Router Gig0/1 → Switch1 (Ventas)
    • Router Gig0/2 → Switch2 (Almacén)
  • Conecta cada PC al switch correspondiente
💡 Nota: Los puertos del router suelen estar en la parte posterior. Haz clic y arrastra el cable desde el switch hasta el puerto correcto del router.

Paso 2: Configurar las PCs

3

Configuración de PC0 (Oficina):

  1. Haz clic en PC0
  2. Selecciona la pestaña Desktop
  3. Haz clic en IP Configuration
  4. Selecciona Static (IP fija)
  5. Ingresa:
    • IP: 192.168.10.10
    • Subnet Mask: 255.255.255.0
    • Gateway: 192.168.10.1
4

Configura las demás PCs:

Dispositivo IP Máscara Gateway
PC1 (Ventas) 192.168.20.10 255.255.255.0 192.168.20.1
PC2 (Almacén) 192.168.30.10 255.255.255.0 192.168.30.1

Paso 3: Configurar el Router (Sin Consola)

5

Configuración de interfaces:

  1. Haz clic en el Router
  2. Selecciona la pestaña Config
  3. Haz clic en Interfaces en el menú izquierdo

Para cada interfaz:

Interfaz IP Máscara Estado
GigabitEthernet0/0 (Oficina) 192.168.10.1 255.255.255.0 ON
GigabitEthernet0/1 (Ventas) 192.168.20.1 255.255.255.0 ON
GigabitEthernet0/2 (Almacén) 192.168.30.1 255.255.255.0 ON
🔄 Importante: Activa cada interfaz cambiando el toggle a ON después de configurar la IP.

Paso 4: Verificar Conexión

6

Pruebas de conectividad:

  1. En cada PC, haz clic en Desktop → Command Prompt
  2. Ejecuta los siguientes comandos:

Desde PC0 (Oficina):

ping 192.168.20.10 # Debe responder (Ventas)
ping 192.168.30.10 # Debe responder (Almacén)

Desde PC1 (Ventas):

ping 192.168.10.10 # Oficina
ping 192.168.30.10 # Almacén
❌ Si el ping falla:
  • Verifica que las IPs y gateways estén correctos
  • Revisa que los cables estén verdes (conectados)
  • Confirma que las interfaces del router estén activadas

🎯 Resultado Esperado

  • ✅ Las 3 PCs pueden comunicarse entre sí
  • ✅ El router enruta correctamente entre redes
  • ✅ No se requirió usar la línea de comandos

Flujo de Comunicación

[Oficina] → [Router] → [Ventas] ↘ [Almacén]
